# DBMS Final Project

## Description
To implement a system that user can communicate with database through GUI.

## Backend
### Install
```
$ pip install pipenv
$ pip install fastapi
$ pip install uvicorn
```

### Setup
```
$ pipenv install
```

### Serve
```
$ pipenv run uvicorn main:app [--reload]
```

## Frontend
### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Compiles and minifies for production
```
npm run build
```
